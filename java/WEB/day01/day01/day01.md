# XML和HTML对比
## XML
- 可扩展标记语言：标签、属性、关系可扩展
- 用来存储或传输数据

## HTML
- 超文本标记语言：标签、属性、关系已固定(w3c)
- 用来显示数据
- 某些版本就是按照XML规范定义的
- 可以将HTML理解为标签固定的XML
